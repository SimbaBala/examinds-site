About TabWell

TabWell is a native macOS source-code editor built for developers moving from Notepad++ on Windows. It brings the core Notepad++ workflow to the Mac: fast multi-tab editing, tree-sitter syntax highlighting, column and multi-caret editing, find in files, and honest encoding handling — all in Swift with TextKit 2.

Requires macOS 14.0 or later. Universal binary — runs natively on Apple Silicon and Intel.
